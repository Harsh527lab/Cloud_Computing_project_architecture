# Screenshots Directory

This directory contains deployment screenshots for documentation.

## Required Screenshots

Please capture the following screenshots during deployment:

### Infrastructure
- [ ] VPC Dashboard showing created VPC
- [ ] Subnet configuration
- [ ] Security Groups list
- [ ] Internet Gateway
- [ ] NAT Gateway
- [ ] Route Tables

### Compute
- [ ] EC2 Instances list
- [ ] Auto Scaling Group configuration
- [ ] Launch Template
- [ ] Application Load Balancer
- [ ] Target Group health checks

### Database
- [ ] RDS Dashboard
- [ ] RDS Instance details
- [ ] DB Subnet Group

### Storage
- [ ] S3 Bucket list
- [ ] S3 Bucket configuration
- [ ] S3 Bucket versioning enabled

### Serverless
- [ ] Lambda Functions list
- [ ] Lambda Function configuration
- [ ] S3 Trigger configuration
- [ ] CloudWatch Logs showing Lambda execution

### Monitoring
- [ ] CloudWatch Metrics
- [ ] CloudWatch Log Groups

### CLI & Boto3
- [ ] Terraform apply output
- [ ] CloudFormation stack creation
- [ ] AWS CLI commands execution
- [ ] Boto3 script outputs

## Naming Convention

Use the following naming format:
`XX_category_description.png`

Example:
- `01_vpc_dashboard.png`
- `02_ec2_instances.png`
- `03_rds_instance.png`
- `04_s3_bucket.png`
- `05_lambda_function.png`
